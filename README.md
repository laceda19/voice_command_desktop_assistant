# 🎙️ Voice Command Desktop Assistant

A Python-based voice assistant that allows users to control their desktop through voice commands. The assistant can recognize speech, speak responses, open applications, launch websites, and perform basic desktop automation.

---

## 📌 Reference

Inspired by:

https://github.com/YahyaSaadME/Windows-Voice-Assistant

* Entry point: `main.py`
* Requires microphone access
* Uses speech recognition and text-to-speech
* Can open applications and websites
* Supports keyboard and mouse automation

---

## ✨ Features

* 🎤 Speech recognition
* 🔊 Text-to-speech responses
* 📂 Open applications
* 🌐 Open websites
* 🖥️ Desktop automation
* ⌨️ Keyboard and mouse control

---

## 🛠️ Technologies Used

* Python
* SpeechRecognition
* pyttsx3
